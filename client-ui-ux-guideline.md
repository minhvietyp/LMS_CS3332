# LMS Client UI/UX Guideline

## Design Direction

The LMS client experience should follow a modern learning-focused UI inspired by:

- F8
- Udemy
- Coursera
- Teachable

The client experience must prioritize:

- learning focus
- progress visibility
- clear navigation
- responsive behavior
- minimal distraction

---

# Course Catalog

## Recommended Layout

- Grid card layout
- Optional list view toggle

## Filters

Use left sticky sidebar filters:

- category
- level
- duration
- rating
- language
- price

## Search

Use:

- global search
- course page search

## Course Card Content

Each course card should contain:

- thumbnail
- title
- instructor
- lesson count
- student count
- rating
- progress bar
- wishlist button
- price/discount badge

---

# Student Dashboard

## Recommended Direction

Use a modern visual dashboard focused on learning progress.

## Dashboard Sections

- Continue Learning
- Recent Lessons
- Upcoming Deadlines
- Quiz Status
- Assignment Status
- Notifications
- Recommended Courses
- Learning Progress

---

# Instructor Dashboard

## Recommended Direction

Instructor pages should use the client learning style, not admin style.

## Dashboard Sections

- Revenue Overview
- Course Analytics
- Student Count
- Recent Enrollments
- Submission Alerts
- Draft Courses
- Pending Reviews

---

# Analytics Charts

## Recommended Libraries

- Ant Design Charts
- Recharts

## Recommended Chart Style

Use:

- minimal charts
- soft colors
- rounded cards

## Recommended Colors

- soft blue
- soft purple
- soft coral

# Quiz UI/UX

## Recommended Layout

Use one question per page.

## Quiz Features

- timer
- question palette
- auto save
- progress indicator
- flag question

# Assignment UI/UX

## Recommended Layout

- submitted

- missing

Use drawer sidebar.

# Theme Direction

## Recommended

Use light mode first.

Dark mode can be implemented later.

---

# Empty States

Use friendly illustration-based empty states for:

- no enrolled course
- no assignments
- no notifications
- no quiz attempts

# UX Direction Summary

## Student Experience

The student experience should feel:

- friendly
- visual
- motivating
- progress-focused

## Instructor Experience

The instructor experience should feel:

- creator-focused
- analytical
- organized

## Admin Experience

The admin experience should feel:

- efficient
- structured
- management-focused
